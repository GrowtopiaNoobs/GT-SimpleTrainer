GT-SimpleTrainer

Simple project to make your own GT trainer!

This project has been developed and tested on Dev-C++ 5.11: https://sourceforge.net/projects/orwelldevcpp/ Also this isn't ready to use trainer! You need to find opcodes, edit and then compile it yourself!
